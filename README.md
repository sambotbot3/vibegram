# Intro

This is the Vibe programming language transpiler.

Write in plain english and program. 
There's no need for any new language now.

Any <>.vibe files you change will be transpiled.

# Quick start
./install-deps.sh
./scripts/build.sh
build/vibegram
build/vibegram /path/to/project

# Test
ctest --test-dir build --output-on-failure
